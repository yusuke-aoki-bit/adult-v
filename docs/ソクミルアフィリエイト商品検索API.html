WEBサービス…商品検索API

このAPIはソクミルアフィリエイト内の商品の情報を取得することが可能なAPIです。

リクエスト仕様

下記のURLにリクエストすることで取得できます。

リクエスト例

https://sokmil-ad.com/api/v1/Item?affiliate_id=47418-001&api_key=70c75ce3a36c1f503f2515ff094d6f60&output=json

パラメーター 要素の説明 必須 値のサンプル 概要
api_key API KEY ○ 70c75ce3a36c1f503f2515ff094d6f60 このページで発行している代理店様固有の値になります。
affiliate_id アフィリエイトID ○ 47418-001 サイト固有の値です。サイト一覧よりご確認ください。
output 出力形式 ○ json JSON：json
xml：xml
hits 取得件数 20 初期値：20　最大：100
offset 取得開始位置 1 初期値：1　最大：50000
sort 並び順 price 商品の並び順です。
初期値：price
価格が高い順：price
価格が安い順：-price
新着：date
category カテゴリ av アダルト動画：av
グラビア：idol
keyword キーワード 桐谷まつり UTF-8で指定してください。商品名/ジャンル名/出演者名から検索します。
article 検索項目 actor 出演者：actor
監督：director
ジャンル：genre
メーカー：maker
レーベル：label
シリーズ：series
article_id 検索ID kiritanimatsuri
gte_date 配信開始日 2016-04-01T00:00:00 指定した日付以降に発売された商品を絞り込むことができます。
ISO8601形式でフォーマットした日付を指定してください。
lte_date 配信開始日 2016-04-01T00:00:00 指定した日付以降に発売された商品を絞り込むことができます。
ISO8601形式でフォーマットした日付を指定してください。
