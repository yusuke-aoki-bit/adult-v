WEBサービス…出演者検索API

このAPIはソクミルアフィリエイト内の出演者の情報を取得することが可能なAPIです。

リクエスト仕様

下記のURLにリクエストすることで取得できます。

リクエスト例

https://sokmil-ad.com/api/v1/Actor?affiliate_id=47418-001&api_key=70c75ce3a36c1f503f2515ff094d6f60&output=json

パラメーター 要素の説明 必須 値のサンプル 概要
api_key API KEY ○ 70c75ce3a36c1f503f2515ff094d6f60 このページで発行している代理店様固有の値になります。
affiliate_id アフィリエイトID ○ 47418-001 サイト固有の値です。サイト一覧よりご確認ください。
output 出力形式 ○ json JSON：json
xml：xml
hits 取得件数 20 初期値：20　最大：100
offset 取得開始位置 1 初期値：1　最大：50000
sort 並び順 price 商品の並び順です。
初期値：price
価格が高い順：price
価格が安い順：-price
新着：date
category カテゴリ av アダルト動画：av
グラビア：idol
gender 性別 f f：女性
m：男性
keyword キーワード 琥珀うた UTF-8で指定してください。商品名/ジャンル名/出演者名から検索します。
initial イニシャル こ UTF-8で指定してください。
bust バスト 88 bust=88ならバスト88cm以上、
bust=-88ならバスト88cm以下、
bust=88-95ならバスト88cm以上～95cm以下
waist ウエスト 60 同上
hip ヒップ 85-90 同上
height 身長 165 同上
cup カップ数 d A～の半角英字
birthday 誕生日 19900101 yyyymmddの形式で指定
birthday=19900101なら1990年1月1日以降生まれ、
birthday=-19900101なら1990年1月1日以前生まれ、
birthday=19900101-19951231なら
1990年1月1日生まれ～1995年12月31日生まれ

レスポンス値

フィールド 要素の説明 値のサンプル 概要
request
├ parameters リクエストした内容をお返しします。
result
├ status ステータスコード 200
├ result_count 取得件数 20
├ total_count 全体件数 150627
├ first_position 検索開始位置 1
└ actor 出演者情報
　├ id 出演者ID kohakuuta
　├ name 女優名 琥珀うた
　├ ruby 女優名（読み仮名） こはくうた
　├ bust バスト 96
　├ category カテゴリ av
　├ gender 性別 female
　├ cup カップ数 C
　├ waist ウェスト 60
　├ hip ヒップ 88
　├ height 身長 168
　├ birthday 生年月日 1987-03-24
　├ imageURL 画像URL http://imgrbn.sokmil.com/pict/staff/100_kohakuuta_T1377832081.jpg
　├ listURL リストページURL https://www.sokmil.com/av/kohakuuta/
　└ affiliateURL アフィリエイトURL https://www.sokmil.com/av/kohakuuta/?affi=10000-001
